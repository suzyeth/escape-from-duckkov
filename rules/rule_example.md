## ⚡ 快速启动命令

### 🎮 `/gd:` 快捷命令（推荐）

> 借鉴 SuperClaude Framework 命令系统，提供标准化的 `/gd:` 前缀快捷命令。
> 详细说明见 `rule_commands.md`。

**需求类命令（启动流水线）**：

| 命令 | 说明 |
|------|------|
| `/gd:new "需求描述"` | 新建需求（进入制作人标准分析流程） |
| `/gd:feature "功能描述"` | 直接按 FEATURE 类型启动 |
| `/gd:feature_ui "功能描述"` | 直接按 FEATURE_UI 类型启动 |
| `/gd:bugfix "问题描述"` | 直接按 BUGFIX 类型启动 |
| `/gd:optimize "优化目标"` | 直接按 OPTIMIZE 类型启动 |
| `/gd:config "配置说明"` | 直接按 CONFIG 类型启动 |
| `/gd:test "测试说明"` | 直接按 TEST 类型启动 |
| `/gd:doc "文档说明"` | 直接按 DOC 类型启动 |
| `/gd:review "审查范围"` | 直接按 REVIEW 类型启动 |
| `/gd:research "调研主题"` | 直接按 RESEARCH 类型启动 |

**管理类命令（不启动流水线）**：

| 命令 | 说明 |
|------|------|
| `/gd:status` | 查看所有需求状态（需求池） |
| `/gd:progress` | 查看当前进度看板 |
| `/gd:resume [REQ-ID]` | 恢复指定/最近的进行中需求 |
| `/gd:plan` | 查看全局策划案摘要 |
| `/gd:tech` | 查看全局技术文档摘要 |
| `/gd:kb` | 查看知识库摘要 |
| `/gd:history [REQ-ID]` | 查看指定需求的完整历史 |
| `/gd:stats` | 查看时长统计数据 |
| `/gd:issues` | 查看问题日志 |

**迭代类命令**：

| 命令 | 说明 |
|------|------|
| `/gd:reflect` | 触发结构化内省 |
| `/gd:iterate "问题描述"` | 触发迭代改进流程 |

### 📝 自然语言命令（兼容保留）

> 以下旧版自然语言命令全部保留兼容，用户可按习惯自由选择。

| 命令 | 说明 | 对应 `/gd:` 命令 |
|------|------|------------------|
| `开发 {功能描述}` | 启动完整流水线 | `/gd:feature "功能描述"` |
| `查看进度` | 查看当前所有需求的进度 | `/gd:progress` |
| `查看需求池` | 查看所有需求状态 | `/gd:status` |
| `查看统计` | 查看时长统计数据 | `/gd:stats` |
| `查看问题` | 查看问题日志 | `/gd:issues` |
| `查看全局策划` | 查看全局策划案 | `/gd:plan` |
| `查看全局技术` | 查看全局技术文档 | `/gd:tech` |
| `继续上次的需求` | 恢复最近的进行中需求 | `/gd:resume` |
| `恢复 REQ-003` | 恢复指定需求 | `/gd:resume REQ-003` |
| `暂停` | 暂停当前流程 | （暂无对应命令） |
| `继续` | 继续执行流程 | （暂无对应命令） |
| `跳过美术` | 跳过美术需求阶段 | （暂无对应命令） |
