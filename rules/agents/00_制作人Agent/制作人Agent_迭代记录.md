# 🎬 制作人 Agent 迭代记录

## 🔍 迭代检查清单

| 检查项 | 触发更新条件 |
|--------|--------------|
| 需求分类准确性 | 分类错误导致流程走错 |
| 关键词识别 | 新关键词无法识别 |
| 流转路径合理性 | 路径不合理导致效率低 |
| 需求类型覆盖 | 出现新的需求类型 |
| **人格一致性** | **出现过度犹豫或插手细节的行为（违反"快速决断"人格）** |
| **🔴 复合需求拆分** | **未识别复合需求导致越权或遗漏** |
| **🔴 产出物归属检查** | **产出物被分配给非对应职责的Agent** |
| **🔴 被纠正后迭代触发** | **被用户纠正后未触发规则迭代** |
