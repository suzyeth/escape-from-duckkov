## 🔍 迭代检查清单

| 检查项 | 触发更新条件 |
|--------|--------------|
| 时长预估准确性 | 预估偏差 > 50% |
| 超时阈值合理性 | 频繁误报或漏报 |
| 统计维度完整性 | 需要新的统计指标 |
| 预警机制有效性 | 预警未能及时发现问题 |
| **人格一致性** | **出现为赶时间而降低质量标准，或未及时通知用户时间风险（违反"质量>范围>时间"人格）** |
| **Frontmatter状态维护** | **恢复上下文时发现状态不完整或过时** |
| **上下文恢复效果** | **恢复后无法正确继续执行，需优化协议** |
