# Unity Test Framework 配置

> 如果项目尚未配置测试框架，需要先添加以下配置

## 1. 创建 Assembly Definition

在 `Assets/Tests/` 目录下创建 `Tests.asmdef`:

```json
{
    "name": "Tests",
    "rootNamespace": "Tests",
    "references": [
        "GUID:你的主程序集GUID"
    ],
    "includePlatforms": [
        "Editor"
    ],
    "excludePlatforms": [],
    "allowUnsafeCode": false,
    "overrideReferences": true,
    "precompiledReferences": [
        "nunit.framework.dll"
    ],
    "autoReferenced": false,
    "defineConstraints": [
        "UNITY_INCLUDE_TESTS"
    ],
    "versionDefines": [],
    "noEngineReferences": false
}
```

## 2. 通过 Package Manager 确保已安装
- `com.unity.test-framework`

## 3. 测试环境

| 项目 | 值 |
|------|-----|
| Unity 版本 | 20XX.X.X |
| 测试平台 | EditMode / PlayMode |
| 操作系统 | Windows 10/11 |
