# Bug 报告模板

> 使用说明：测试失败时，按此格式生成Bug报告

```markdown
## 🐛 BUG-{编号}

| 属性 | 值 |
|------|-----|
| Bug ID | BUG-001 |
| 关联测试 | AC001_XXX |
| 关联验收标准 | AC-001 |
| 严重程度 | P0 (阻塞) / P1 (严重) / P2 (一般) |
| 当前状态 | 待修复 |
| 修复轮次 | 第1轮 |

### 问题描述
[详细描述问题现象]

### 复现步骤
1. [步骤1]
2. [步骤2]

### 期望结果
[应该是什么]

### 实际结果
[实际是什么]

### 错误信息
[错误日志/堆栈]

### 定位分析
[分析可能的原因]

### 修复建议
[给程序 Agent 的建议]
```
