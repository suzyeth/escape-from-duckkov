# 逃离鸭科夫

使用 Three.js 构建的俯视角撤离射击游戏 — Q版卡通外表下的硬核战术玩法。

**[在线试玩](https://suzyeth.github.io/escape-from-duckkov/)**

## 简介

逃离鸭科夫是一款浏览器端撤离射击游戏。你会被投放到一张充满敌意的地图中，搜刮物资、击杀AI敌人（鸭卒），并在时间耗尽前成功撤离。死亡意味着失去一切。

灵感来自《逃离塔科夫》，但采用 Q 版卡通美术风格 — 圆润的角色、暖色光照、可爱的视觉与残酷的机制形成反差。

## 游戏特性

- **5 种武器** — AK-74、格洛克17、MP-133霰弹枪、VSS微声狙击、MP5冲锋枪
- **5 种敌人** — 鸭卒、精英鸭卒、暴走鸭、重甲鸭、BOSS鸭王
- **部位伤害** — 头/躯干/双臂/双腿，独立血量，骨折与流血机制
- **护甲系统** — 轻甲、重甲、头盔，有耐久度
- **网格背包** — 8×10格，负重限制，物品堆叠
- **撤离系统** — 3个撤离点，45分钟突袭时限，按住E键10秒撤离
- **天赋系统** — 12个永久升级（生命、移速、精准、护甲等）
- **制造系统** — 8个配方，部分需要蓝图解锁
- **战争迷雾** — 120°视野锥，视野外敌人不可见
- **波次系统** — 每90秒刷新一波敌人
- **难度选择** — 简单/普通/困难
- **完整HUD** — 血条、弹药、快捷栏、小地图、Boss血条、受击方向指示

## 技术栈

- **Three.js** — 3D渲染，PBR材质，阴影，泛光后处理
- **Vite** — 构建工具
- **lil-gui** — 实时美术参数调节
- 全程序化几何体 — 所有道具模型由 Three.js 基础形体构建，无外部模型文件
- ACES色调映射、顶点色地形、粒子特效

## 本地开发

```bash
cd fps-game
npm install
npm run dev     # http://localhost:5173/escape-from-duckkov/
npm run build   # 输出到 dist/
```

## 项目结构

```
fps-game/src/
├── main.js              # 游戏主循环、输入、射击、UI接线
├── core/                # 渲染器、游戏循环、输入管理
├── entities/            # 玩家、敌人、远程玩家
├── systems/             # 武器、子弹、AI、血量、背包、战利品、
│                        # 迷雾、门、音效、天赋、制造、网络
├── props/               # 程序化3D模型（28种战利品、
│                        # 树木、岩石、围栏、建筑、车辆）
├── ui/                  # HUD、背包UI、小地图、仓库界面
└── world/               # 关卡（地图、区域、光照）
```
